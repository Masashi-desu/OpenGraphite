導入方法 OpenGraphite をローカルで使い始める。 リポジトリを取得し、ローカルで build して、同梱の sample project を開きます。下の手順に沿って、準備から OpenGraphite の起動確認まで進められます。 リポジトリを開く 手順を見る 1. リポジトリを取得する 作業用フォルダで OpenGraphite を clone し、リポジトリに移動します。以降のコマンドはこの場所で実行します。 2. build して sample を確認する 必要に応じて XcodeGen を入れ、app と CLI を build します。build が終わったら validation を一度実行し、同梱 sample を開ける状態か確認します。 3. sample project を開く OpenGraphite.app を起動し、Welcome 画面で Open Sample Project を選びます。同梱ページが開くので、すぐに表示を確認して編集できます。
